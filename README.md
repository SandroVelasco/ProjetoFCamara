## Requerimentos

- Git
- NodeJS

## Clone do projeto

```
git clone https://github.com/SandroVelasco/ProjetoFCamara.git
```

## Instalação das dependências

```
npm install
```

## Como iniciar o site

```
node index.js
```

## URL Local

```
http://localhost:3000
```
